Problem

Can an agent stay fair under pressure?

This environment trains judgment, not recall. The agent must detect fabricated evidence, resist manipulation, and still issue a defensible ruling.

Environment

Read, challenge, and rule inside a live hearing

Each case exposes statements, evidence, manipulation signals, and a strict step budget. The agent can inspect, question, and then rule.

Reward Design

More than verdict accuracy

Scoring combines verdict quality, fabricated-evidence detection, precedent consistency, and manipulation resistance so the model cannot game one shortcut.

Training Proof

Real OpenEnv loop, not a static prompt demo

The repo includes a runnable training script, committed reward plots, and a notebook so judges can inspect the learning pipeline rather than just the UI.
